## Descrição

Este projeto é codificado em Java e utiliza o framework [Spring](<https://spring.io/>).



## Requerimentos

<p>Devem estar instaladas e configuradas as seguintes ferramentas:</p>
<ul>
   <li>Java 11</li>
   <li>Maven 3.8.1</li>
   <li>Docker ou Podman</li>
</ul>
